# Enterprise Code Intelligence Platform (ECIP)

# Project Roadmap

Version: 1.0

Target Framework: .NET 8 LTS

---

# Vision

Build an Enterprise AI-powered Code Intelligence Platform capable of understanding enterprise source code repositories using deterministic analysis combined with Agentic AI.

The project demonstrates enterprise architecture, AI integration, RAG, Prompt Registry, MCP, AI Gateway and software engineering best practices.

---

# Development Strategy

Each sprint must produce:

- Working application
- Successful build
- Git commit
- Demoable functionality

Never leave the solution in a broken state.
